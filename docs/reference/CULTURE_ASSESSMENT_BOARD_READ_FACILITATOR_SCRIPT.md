# Loep Culture Assessment Board-Read Facilitator Script

## Duration
60-90 minutes

## Goal
Guide a board-level read of the annual culture and engagement baseline as a spacious, governed conversation. Keep the deck guided, calm, and bounded so the room does not drift into benchmark reading, ranking logic, or self-serve analysis behavior.

## Slide Order and Presenter Rhythm
1. Opening / purpose
- Open with tone, scope, and seriousness.
- Introduce the deck as a guided annual baseline discussion.
- Set the expectation that interpretation stays inside explicit governance boundaries.

2. What was measured
- Clarify the fixed-method baseline before discussing any signal.
- Explain that this route uses ten fixed domains and is not a build-your-own survey.
- Keep this slide concise and confidence-building.

3. Response and governance
- Establish trust in the response base.
- Present governance as quality control, not caution tape.
- Say clearly what may be shown, what stays hidden, and why that is methodologically appropriate.

4. Executive read
- Read the core executive line aloud.
- Use it to frame the rest of the deck, not to replace later evidence.
- Keep the tone descriptive and bounded rather than diagnostic.

5. Culture Index hero
- Present the `Loep Culture Index` as a navigation signal.
- State explicitly that it is not a pass/fail verdict, gauge, or benchmark position.
- Connect it to the domain picture that follows.

6. Domain picture
- Walk the room through the ten-domain profile as a reading sequence.
- Avoid highest/lowest rhetoric.
- Use one visible pattern to orient the next discussion step.

7. Board attention point 1
- Give the first attention point its own full discussion moment.
- Explain why it matters now and what should be verified next.
- Keep the wording descriptive rather than causal.

8. Board attention point 2
- Treat the second attention point as a distinct board topic.
- Link it back to the broader pattern already shown.
- Avoid blame or intervention theater.

9. Board attention point 3
- Close the attention set cleanly with the third issue or tension.
- Keep the confidence language bounded to stability, not certainty.
- Prepare the room for a governed contrast view only if release allows it.

10. Segment contrast, if released
- Show segment differences only where release status permits.
- If contrast is not released, say so plainly and present that as a governance state rather than an omission.
- Do not allow best/worst segment framing or league-table language.

11. What not to conclude
- Use this slide to actively prevent overclaim.
- Present the boundaries calmly: no causal proof, no manager ranking, no individual inference, no benchmark claim.
- Frame this as quality discipline, not defensiveness.

12. Decision questions
- Use the decision questions to open discussion, not to prescribe action.
- Keep the questions tied to the baseline evidence actually shown.
- Let the board clarify priorities without turning the deck into a task-management tool.

13. Follow-on route options
- End with bounded route choices after the baseline.
- Explain that no route opens automatically.
- Present Pulse or deeper governed work only as explicit follow-on options, not as default continuation.

## Facilitator Guardrails
- keep each slide to one dominant speaking line
- keep each attention point as a separate discussion moment
- keep governance visible throughout the read
- keep the boardroom pace spacious rather than dense
- do not turn the index into a health verdict
- do not introduce benchmark, ranking, or pass/fail language
- do not speculate about individual managers or unreleased layers
- do not promise cause-and-effect or imply an automatic next step
