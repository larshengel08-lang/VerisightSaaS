# Loep Culture Assessment Board-Read Facilitator Script

## Duration
60-90 minutes

## Goal
Guide a board-level read of the annual culture and engagement baseline as a spacious, governed conversation. Keep the deck guided, calm, and bounded so the room does not drift into benchmark reading, ranking logic, or self-serve analysis behavior.

## Canonical Inheritance
The canonical slide sequence, slide titles, and fixed boundary language inherit directly from `CULTURE_ASSESSMENT_BOARDROOM_DECK.md`.

Use this script as presenter guidance for that deck, not as a parallel content specification.

Fixed boundary language for the full read:
- no causal proof
- no manager ranking
- no individual inference
- no benchmark claim

## Presenter Rhythm Across the Canonical 13-Slide Sequence
1. Opening / purpose
- Set tone, scope, and seriousness.
- Frame the session as guided, annual, and explicitly bounded before any result is read.

2. What was measured
- Clarify the fixed-method baseline briefly and confidently.
- Keep this as method scope, not a second content deck inside the deck.

3. Response and governance
- Establish trust in the response base.
- Explain governance as quality control and visibility logic.

4. Executive read
- Read the main line aloud.
- Position the remaining slides as an unpacking of that line.

5. Culture Index hero
- Present the index as orientation, not verdict.
- Keep the room out of pass/fail or benchmark language.

6. Domain picture
- Walk the sequence as a guided read.
- Avoid top/bottom or winner/loser rhetoric.

7. Board attention point 1
- Give this point its own discussion moment.
- Stay descriptive and bounded.

8. Board attention point 2
- Keep the second point distinct from the first.
- Connect it back to the pattern already shown.

9. Board attention point 3
- Close the attention set cleanly.
- Prepare the room for governed contrast only if release allows it.

10. Segment contrast, if released
- Show released differences only.
- If not released, say so plainly and treat that as a governance state.

11. What not to conclude
- Read the fixed boundary language calmly and directly.
- Present it as quality discipline, not defensiveness.

12. Decision questions
- Open discussion without prescribing action.
- Keep the questions tied to evidence already shown.

13. Follow-on route options
- End with bounded route choices only.
- Do not imply automatic continuation.

## Facilitator Guardrails
- keep each slide to one dominant speaking line
- keep each attention point as a separate discussion moment
- keep governance visible throughout the read
- keep the boardroom pace spacious rather than dense
- do not turn the index into a health verdict
- do not introduce benchmark, ranking, or pass/fail language beyond the inherited boundary framing
- do not speculate about individual managers or unreleased layers
- do not promise cause-and-effect or imply an automatic next step
