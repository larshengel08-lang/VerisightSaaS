# Loep Culture Assessment Sample Invite Mail

Subject: Uitnodiging - Loep Cultuurbeeld

Beste collega,

Je bent uitgenodigd voor Loep Cultuurbeeld, de jaarlijkse organisatiebrede baseline voor cultuur en engagement.

Deze meting helpt om op organisatieniveau zichtbaar te maken welke brede patronen aandacht vragen. De uitkomst wordt gelezen op geaggregeerd niveau, met duidelijke privacy- en governancegrenzen.

Praktisch:
- invullen duurt ongeveer 10-12 minuten
- de vragenlijst is vertrouwelijk
- individuele antwoorden worden niet als losse beoordeling gebruikt

Alvast bedankt voor je bijdrage.
