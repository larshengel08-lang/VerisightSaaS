# Loep Culture Assessment - What You Receive After Completion

- Board report PDF
- Guided board-read / boardroom PDF deck

Optional or governed outputs are not part of the standard baseline package:

- Optional HR appendix in governed package
- Optional segment summary export in governed package
- Optional Pulse follow-on after the annual baseline
