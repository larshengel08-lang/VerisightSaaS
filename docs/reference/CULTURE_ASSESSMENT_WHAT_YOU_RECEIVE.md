# Loep Culture Assessment - What You Receive After Completion

What you receive after completion stays bounded to one premium, board-first output family:

## Standard baseline package

- Board report PDF
  - compact and self-readable
  - bounded to the annual baseline
  - no benchmark, ranking, or causal framing

- Guided board-read / boardroom PDF deck
  - more spacious and discussion-led than the report
  - built for facilitated board-read
  - same governance and interpretation boundaries as the report

- Executive one-pager
  - later derivative of the same output family
  - not the first pilot artifact
  - no extra analysis layer beyond the board baseline

Optional or governed outputs are not part of the standard baseline package:

- Optional HR appendix in governed package
- Optional segment summary export in governed package
- Optional Pulse follow-on after the annual baseline

## What this package does not do

- no benchmark ranking
- no self-serve builder
- no custom questionnaire modules
- no named manager layer by default
- no individual respondent export
- no manager defense pack
- no hidden segment leakage

## Sample Output v2 reference

The first upgraded sample package is built for the fictive organization `Noordhaven Industrie Groep` and must show:

- a moderate and nuanced Culture Index
- ten domains in bestuurlijke leesvolgorde
- three board attention points
- two recurring theme pairs
- two safe released segment contrasts
- one hidden segment shown as governance state
- only safe text cluster summary, never raw quotes
- Pulse only as a conditional follow-on

## Buyer-facing promise

After completion, the client receives:

- one bounded board report PDF
- one guided boardroom deck
- one explicit governance frame
- optional governed deepening only when released

The client does not automatically receive:

- benchmark packs
- self-serve analytics workspace
- manager score comparisons
- unrestricted segment exports
