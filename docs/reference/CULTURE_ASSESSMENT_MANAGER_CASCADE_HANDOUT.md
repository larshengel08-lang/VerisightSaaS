# Loep Culture Assessment Manager Cascade Handout

## This Handout May
- explain what was measured at organization level
- support safe team communication
- structure a follow-up conversation without score defense

## This Handout May Not
- compare teams as a ranking
- ask managers to defend low scores
- discuss individual signals
- show local scores below threshold

## Manager Message
Start from broad organization patterns, then ask what local context might help explain them. Stay descriptive and avoid blame.
