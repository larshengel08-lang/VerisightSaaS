## 2. Shared grammar

### Insight grammar

Gebruik suitebreed:

- signalen
- managementread
- eerste managementvraag
- prioriteit
- first value
- first management use

Niet suitebreed gebruiken als hoofdtaal:

- diagnose
- voorspelling
- causaliteit
- performance score

### Follow-through grammar

Gebruik suitebreed:

- eigenaar
- eerste stap
- assignment
- reviewmoment
- follow-through
- closeout
- bounded vervolg

Niet suitebreed gebruiken als hoofdtaal:

- taakplatform
- orchestration layer
- cockpitproduct
- AI actie-engine

### Access grammar

Gebruik suitebreed:

- owner
- member
- viewer
- manager assignee
- bounded scope
- denied state

## 3. Cross-surface rules

### Dashboard

Dashboard spreekt vanuit:

- signalen
- prioriteiten
- managementread

### Report

Report spreekt vanuit:

- managementsamenvatting
- bestuurlijke handoff
- eerste vervolgrichting

### Action Center

Action Center spreekt vanuit:

- follow-through
- assignment
- review
- closure

### Site

De site spiegelt dit als:

- people insights
- bepalen wat eerst telt
- acties expliciet maken
- acties toewijzen
- review en follow-through bewaken
