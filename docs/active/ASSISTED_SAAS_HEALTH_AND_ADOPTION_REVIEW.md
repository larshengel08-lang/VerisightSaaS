# ASSISTED_SAAS_HEALTH_AND_ADOPTION_REVIEW.md

Last updated: 2026-04-26
Status: active
Source of truth: delivery records, onboarding acceptance, Action Center operating signals and the current suite shell.

## 1. Summary

Dit document definieert de minimale health- en adoptionreview voor Verisight als assisted suite.

Doel:

- frictie vroeg zien
- access- en activationproblemen snel zien
- first value en follow-through niet alleen als productpolish, maar als operating signal behandelen

## 2. Weekly health review

Minimaal wekelijks toetsen:

- open contactrequests zonder heldere route
- campaigns zonder complete deliveryrecord/checkpoints
- activation pending zonder opvolging
- first value nog niet bereikt
- first management use nog niet bevestigd
- Action Center dossiers met:
  - owner missing
  - review due
  - blocked assignment
  - decision due

## 3. Monthly adoption review

Maandelijks toetsen:

- hoeveel organisaties actief zijn
- hoeveel campaigns echt first value bereikten
- hoeveel campaigns first management use haalden
- hoeveel Action Center-dossiers echt closeout kregen
- hoeveel manager assignments actief gebruikt werden
- waar frictie terugkerend zit:
  - provisioning
  - activation
  - dashboard read
  - report handoff
  - manager-only access
  - follow-through

## 4. Minimal signal set

De minimale assisted SaaS signalen zijn nu:

- organization created
- owner access confirmed
- campaign active
- import cleared
- invites live
- client activation confirmed
- first value confirmed
- first management use confirmed
- Action Center dossier opened
- Action Center review scheduled
- Action Center closeout decided

## 5. What this is not

Dit is nog niet:

- een volledige analytics stack
- seat- of billing reporting
- cohort-SaaS telemetry
- product-led growth instrumentation

Dit is wel:

- een herhaalbare minimumlaag voor assisted health review

## 6. Validation

- Sluit aan op de huidige suite-shell en accessmodel.
- Sluit aan op current customer ops en Action Center operating model.
- Claimt geen analytics maturity die de runtime nog niet draagt.

## 7. Next step

Gebruik deze reviewlaag als input voor:

- operations closeout
- suite governance
- assisted SaaS launch verdict
