# Loep Cultuurbeeld Board-Read Agenda

Standaardduur:
60-90 minuten

Deelnemers:
- board / directie
- HR lead of People lead
- Loep / Verisight facilitator

Vaste agenda:
1. Context, doel en governancegrens
2. Wat is gemeten en hoe breed is de responsbasis
3. Executive culture read
4. Loep Culture Index als navigatiesignaal
5. Domeinbeeld en samenhang
6. 3-5 board attention points
7. Veilige segmentcontrasten
8. Wat expliciet niet geconcludeerd mag worden
9. Board-beslissingsvragen
10. Vervolgroute en ritme

Expliciet niet:
- vrije score-interpretatie
- manager ranking
- individuele inference
- causale diagnose
