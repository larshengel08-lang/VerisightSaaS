# Loep Cultuurbeeld Demo Environment

Doel:
Interne gids voor een begeleide demo van Loep Cultuurbeeld zonder echte klantdata.

Gebruik de demo-omgeving alleen met:
- fictieve organisatie
- fictieve respondenten
- gesloten baseline
- governed drilldown alleen wanneer veilig

Demo-volgorde:
1. Laat de productkaart en routepositionering zien
2. Open de campaign-detailpagina van de fictieve baseline
3. Toon de executive block order
4. Open het boardrapport
5. Toon optioneel de governed segmentexport
6. Eindig met board-read, executive one-pager en what-you-receive artefacts

Niet doen:
- framen als live klantomgeving
- benchmark suggereren
- managerdetail openen
- custom questions of self-serve onboarding suggereren
