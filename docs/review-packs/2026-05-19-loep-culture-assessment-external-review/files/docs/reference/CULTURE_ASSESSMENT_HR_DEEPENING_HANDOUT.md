# Loep Culture Assessment HR Deepening Handout

## Role of HR
HR is the implementation partner, not the owner of isolated score defense.

## Use This Layer For
- preparing the board-read
- checking safe segment contrasts
- structuring governed follow-up
- deciding whether a bounded drilldown is justified

## Do Not Use This Layer For
- ranking teams or managers
- defending a score as good or bad
- claiming causality
- opening named manager detail by default

## Governance Notes
- organization minimum-n applies first
- segment comparison opens only above threshold
- open text appears only when clustering is safe
- no individual respondent export
