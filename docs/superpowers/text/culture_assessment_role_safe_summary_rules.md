# Loep Culture Assessment Role-Safe Summary Rules

**Date:** 2026-05-20  
**Status:** text-governance summary artifact  
**Primary scope:** `Loep Culture Assessment / Loep Cultuurbeeld`  
**Canonical route id:** `culture_assessment`

---

## 1. Purpose

This artifact defines what kind of text-derived summary each audience may receive.

---

## 2. Executive Rules

Executive text output may include:

- short cluster-level summaries
- broad recurring pattern wording
- confidence-bounded contextual signal

Executive text output may not include:

- raw quotes
- local complaint detail
- person-level inference
- blame wording

---

## 3. HR Governed Rules

HR text output may include:

- deeper cluster explanation
- grouped summary themes
- hidden-text reason model
- governed contextual follow-up cues

HR text output may not include:

- unsafe raw dumps
- thin identifiable subgroup detail
- unsupported causal claims

---

## 4. Manager Cascade Rules

Manager cascade text may include:

- optional safe wording for communication support only

Manager cascade text may not include:

- raw quote content
- attributed complaint language
- local score defense prompts
- local blame cues

---

## 5. Summary Language Rule

Allowed language:

- descriptive
- confidence-bounded
- contextual

Forbidden language:

- causal proof
- sentiment verdict
- manager rating
- individual prediction

---

## 6. Approval

Approved when:

- executive rules are explicit
- HR rules are explicit
- manager cascade rules are explicit
- summary language boundaries are explicit
