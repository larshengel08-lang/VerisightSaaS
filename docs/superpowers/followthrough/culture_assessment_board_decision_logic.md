# Loep Culture Assessment Board Decision Logic

**Date:** 2026-05-20  
**Status:** follow-through decision artifact  
**Primary scope:** `Loep Culture Assessment / Loep Cultuurbeeld`  
**Canonical route id:** `culture_assessment`

---

## 1. Purpose

This artifact defines the fixed board decision logic after the board-read.

It exists to prevent:

- vague post-read drift
- premature intervention claims
- local blame disguised as executive decision making

---

## 2. Canonical Board Decision Questions

The board-read must end with these five questions:

1. Which one or two themes require board sponsorship rather than local interpretation?
2. Which visible patterns should remain hypotheses rather than conclusions?
3. Which follow-up questions belong to HR governed deepening rather than immediate action?
4. Which cascade messages are safe to communicate now, and which are not?
5. Is any follow-on route justified now, or is no immediate next route the better decision?

---

## 3. What Not To Conclude

The board may not conclude:

- which manager is the problem
- what the single root cause is
- which intervention is automatically required
- that deeper local action is safe simply because the board wants it

---

## 4. Sponsorship Boundary

Board sponsorship is appropriate for:

- executive priority-setting
- governance-safe follow-up direction
- confirming whether the route needs deeper HR work, bounded follow-through, or no immediate next route

Board sponsorship is not appropriate for:

- local score defense
- team-level ranking
- instructing unsupported causal conclusions

---

## 5. Approval

Approved when:

- the five canonical decision questions are explicit
- non-causal guardrails are explicit
- sponsorship boundaries are explicit
