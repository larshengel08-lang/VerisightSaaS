## 1. Purpose

This document translates the `Loep Culture Assessment` operating system into a **category capability benchmark**.

Its purpose is not to make Loep copy Gallup, Perceptyx, Culture Amp, or Qualtrics feature-by-feature.

Its purpose is to answer:

- which capabilities are table stakes in the same buying context
- which capabilities Loep must build to be taken seriously
- which capabilities Loep should explicitly refuse to imitate
- which capabilities are launch-critical now versus later maturity work

This benchmark should be used as the control layer for:

- roadmap prioritization
- stream gap-maps
- spec sequencing
- pilot-readiness decisions
- launch/no-launch decisions for enterprise-facing claims

## 5. Current Comparative Verdict

### 5.1 Where Loep is already directionally strong

- clear board-level wedge
- governance discipline
- premium baseline logic
- clear refusal of manager ranking and self-serve sprawl
- strong route boundaries with Pulse, RetentieScan, and ExitScan

### 5.2 Where Loep is still materially behind

- methodology authority as a named, trustable asset
- program operations maturity
- action and follow-through systematization
- text intelligence maturity
- enterprise trust and procurement readiness
- repeatable premium delivery at volume

### 5.3 Where Loep should refuse to imitate

- full workflow breadth of Qualtrics
- broad people-platform identity of Culture Amp
- benchmark-first posture
- loose manager analytics and ranking behavior
