# Action Center Minimum Viable Route Package Template

## Status
Template

## Purpose

Provide the minimum fillable package for instantiating an admitted Action Center route inside an already-approved route family.

This template is an operational record that points back to the universal operating contract. It does not restate or weaken that contract, and it does not authorize route expansion, launch, pricing, GTM, or broader operating behavior.

## How To Use This Template

Complete every section below. If a section cannot be completed cleanly, the route is not ready for activation.

## 7. Stop Rules

Mark the route package as blocked if any item below is true:

- the route needs more than one management question
- the route cannot explain closeout cleanly
- the route needs recurring evidence exceptions
- the route creates employee monitoring pressure
- the route reads like a personnel dossier
- the route implies a broader work system

### Retirement Trigger Record

If a stop rule is hit after activation, record:

- trigger observed
- date observed
- who reviewed it
- retirement or deactivation decision

Contract reference:

- Section 16. Route Deactivation Or Retirement Rule
