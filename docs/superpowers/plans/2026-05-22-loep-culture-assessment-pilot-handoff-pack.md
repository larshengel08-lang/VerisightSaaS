## 2. Required Pilot Handoff Materials

Minimum handoff pack:

- board report PDF
- boardroom PDF deck
- board-read facilitator script
- board-read agenda
- launch readiness checklist
- closure and release checklist
- buyer FAQ
- privacy, export, and release pack
- operating owner map
- pilot gate read

Optional, not required for pilot go:

- executive one-pager
- HR appendix where not yet explicitly allowed
- deeper governed handouts not enabled for the bounded pilot

- real named owners replace the fictive placeholders
