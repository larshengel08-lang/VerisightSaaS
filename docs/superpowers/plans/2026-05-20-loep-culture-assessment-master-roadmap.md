# Loep Culture Assessment Master Roadmap

> **For agentic workers:** This roadmap is a strategic execution index for the `Loep Culture Assessment` operating system. Use it to decide which spec or plan must exist next. Do not treat this as one implementation plan.

**Goal:** move `Loep Culture Assessment / Loep Cultuurbeeld` from a strong baseline-only primary route into a premium boardroom assessment with enterprise-grade governance, top-tier output quality, and credible maturity in the buying context of Gallup, Perceptyx, Culture Amp, and Qualtrics.

**Architecture:** this roadmap is organized by capability stream, not by calendar. Each stream includes current state, strengths, enterprise gaps, top-tier gaps, phase movement, and the next required artifacts. Execution must remain bounded per stream.

**Tech Stack / Product Surface:** Loep suite product logic, frontend result environment, backend scoring and report contracts, premium delivery assets, governance controls, planning/spec documentation.

---

## 1. Roadmap Rules

This roadmap follows five rules:

1. `Loep Culture Assessment` remains one product with one core route id: `culture_assessment`
2. enterprise and MKB use the same core logic, with deployment-profile differences only
3. no benchmark-first, no self-serve platform sprawl, no manager ranking, and no individual prediction
4. every next artifact must have explicit acceptance criteria
5. no stream should jump ahead of its dependencies

---

## 2. Current Overall Read

Current product state:

- the route exists as a real baseline-only primary route
- the questionnaire and scoring lock materially exist
- the board-facing framing is strong
- premium delivery is partially real and partially blueprint-based
- governance is directionally strong but not yet enterprise-complete
- drilldown, text intelligence, trust readiness, and listening operations are not yet mature enough for top-tier parity

Current maturity summary:

- overall product phase: `A`
- strongest areas: governed product stack, board framing, route identity
- weakest launch-sensitive areas: methodology authority, trust/procurement readiness, listening operations
- weakest top-tier areas: text intelligence, action/follow-through, benchmark readiness

## 6. Acceptance Rules for Every Next Artifact

Every artifact created from this roadmap must define:

- what "done" means
- who approves it
- whether it is launch-critical
- which next artifact it unlocks

If any artifact does not define those four things, it is incomplete.

---

## 7. Roadmap Verdict

The product is strong enough to justify continued investment.

It is not yet strong enough to claim top-tier category maturity.

The next correct move is not broad implementation.
The next correct move is:

- tighten the first-wave specs
- tighten the first-wave acceptance bars
- then execute in bounded streams

That is the fastest route to making `Loep Culture Assessment / Loep Cultuurbeeld` feel serious in the same buying context as Gallup, Perceptyx, Culture Amp, and Qualtrics while keeping the Loep wedge intact.
