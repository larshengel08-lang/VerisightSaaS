**Date:** 2026-05-20  
**Status:** operations cadence artifact  
**Primary scope:** `Loep Culture Assessment / Loep Cultuurbeeld`  
**Canonical route id:** `culture_assessment`

---

## 1. Default Cadence

- `T-3`: optional pre-announcement
- `T0`: invite
- `T+3 / T+4`: first reminder
- `T+7`: second reminder
- `T-2 before close`: final reminder

Close date stays fixed unless the release owner approves an extension.

---

## 2. Monitoring Rule

Response monitoring must be active enough to identify:

- insufficient coverage risk
- segment ambition that has become unsafe
- whether reminder cadence should continue as planned
